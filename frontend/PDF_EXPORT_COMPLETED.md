## 🚀 使用方法

### 1. 编辑简历
在编辑器页面填写完整信息

### 2. 选择模板
点击"导出"按钮 → 选择模板 → 经典/现代

### 3. 导出 PDF

**方式一：下载**
- 点击"下载 PDF (经典)" 或 "下载 PDF (现代)"
- PDF 自动下载到本地

**方式二：预览**
- 点击"预览 PDF (经典)" 或 "预览 PDF (现代)"
- 在新窗口中打开预览
- 可在浏览器中打印或保存

## 📦 技术实现

### 依赖库
```json
{
  "@react-pdf/renderer": "^4.2.0"
}
```

### 核心 API
```typescript
import { pdf } from '@react-pdf/renderer';
import { Document, Page, Text, View, StyleSheet } from '@react-pdf/renderer';

// 生成 PDF
const blob = await pdf(<PDFDocument resume={resume} />).toBlob();

// 下载
const url = URL.createObjectURL(blob);
const link = document.createElement('a');
link.href = url;
link.download = 'resume.pdf';
link.click();

// 预览
window.open(url, '_blank');
```

### 中文字体支持
使用 Google Fonts 的 Noto Sans SC：
```typescript
Font.register({
  family: 'Noto Sans SC',
  src: 'https://fonts.gstatic.com/s/notosanssc/v36/...'
});
```
